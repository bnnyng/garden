>[!example] Definition: Bounded set
>A set $E \subset X$ is **bounded** if there exists a point $x \in X$ and some $r > 0$ such that $E$ is contained in the $r$-ball about $x$ given by 
>$$
>B(x, r) = \{y \in X \ | \ d(x,y) < r \}.
>$$
