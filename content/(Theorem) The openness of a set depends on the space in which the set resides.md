>[!abstract] Theorem
>Let $E \subset Y \subset X$. The subset $E \subset Y$ is [[Open sets|open]] in the [[Metrics, metric spaces, and the metric topology|metric space]] $(Y, d)$ if and only if $E = Y \cap U$ for some open set $U$ of $(X, d)$.
